## 📖 **نبذة عن المشروع**

**متجر الحلويات اللذيذة** هو موقع ويب احترافي متكامل لعرض وبيع الحلويات المتنوعة. يجمع المشروع بين تصميم عصري جذاب وتجربة مستخدم سلسة مع ميزات متقدمة.

### **المميزات الرئيسية:**
✅ عرض 10 أنواع مختلفة من الحلويات  
✅ نظام بحث وتصفية متقدم  
✅ عربة تسوق ديناميكية مع حساب الأسعار  
✅ وضع Dark Mode / Light Mode  
✅ نموذج اتصال مع التحقق من البيانات  
✅ Responsive Design (جوال وتابلت وسطح مكتب)  
✅ تخزين البيانات في Local Storage  
✅ واجهة مستخدم احترافية وسهلة الاستخدام  

---

## 🛠️ **الأدوات والتقنيات المستخدمة**

### **Frontend:**
- **HTML5** - هيكل دلالي احترافي
- **CSS3** - تصميم متقدم مع Grid و Flexbox
- **JavaScript (Vanilla)** - تفاعلات ديناميكية بدون مكتبات خارجية

### **Features:**
- 🔍 Search functionality - البحث عن المنتجات
- 🎯 Filter by category - تصفية حسب الفئة
- 🛒 Shopping Cart - عربة تسوق متقدمة
- 💾 Local Storage - تخزين البيانات محلياً
- 🌙 Dark Mode Toggle - تغيير الوضع
- ✅ Form Validation - التحقق من صحة النماذج
- 📱 Responsive Design - تصميم متجاوب

## 🚀 **كيفية الاستخدام**

### **1. فتح الموقع:**
```bash
# على جهازك المحلي
1. افتح الملف index.html في المتصفح
أو
2. استخدم Live Server على VS Code
```

### **2. التصفح والبحث:**
- استخدم خاصية البحث 🔍 للبحث عن المنتجات
- استخدم أزرار التصفية لعرض منتجات معينة

### **3. الشراء:**
- أضف المنتجات للسلة بالضغط على "أضف للسلة"
- عدّل الكمية إذا أردت
- اضغط "تأكيد الطلب" لإكمال الشراء

### **4. الميزات الإضافية:**
- 🌙 اضغط على الزر في الأعلى لتبديل الوضع الليلي
- 📞 استخدم نموذج الاتصال للتواصل معنا

## 💻 **تفاصيل الكود**

### **HTML Structure:**
```html
- Header مع Navigation Menu
- Hero Section جذاب
- Search & Filter Section
- Products Grid (10 منتجات)
- Shopping Cart Section
- About Section
- Contact Form مع Validation
- Footer
```

### **CSS Features:**
```css
- CSS Variables للألوان والخطوط
- Grid Layout للمنتجات
- Flexbox للتخطيطات
- Media Queries للـ Responsive Design
- Animations و Transitions
- Dark Mode Support
```

### **JavaScript Functions:**
```javascript
- renderProducts() - عرض المنتجات
- addToCart() - إضافة للسلة
- removeFromCart() - حذف من السلة
- handleSearch() - البحث
- handleFilter() - التصفية
- toggleTheme() - تبديل الوضع
- handleFormSubmit() - معالجة النموذج
- saveCartToStorage() - حفظ في Local Storage
```
