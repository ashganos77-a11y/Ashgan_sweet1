# 🍬 متجر الحلويات اللذيذة (Sweet Delight Shop)

## 📝 فكرة المشروع:
موقع ويب تفاعلي وبسيط لمتجر حلويات مصغر، يتيح للمستخدم تصفح قائمة من الحلويات المتنوعة وإضافتها إلى سلة المشتريات، حيث يتم تحديث المجموع الكلي للأسعار تلقائياً مع إمكانية تأكيد الطلب.

## 🛠️ الأدوات المستخدمة:
* **HTML5:** لبناء الهيكل الأساسي وتنظيم عناصر الصفحة بشكل صحيح.
* **CSS3:** لتنسيق الألوان، الخطوط، وتصميم واجهة جذابة ومتناسقة.
* **JavaScript:** لإضافة التفاعل الحيوي مثل تشغيل سلة المشتريات وحساب الأسعار.

## 📁 هيكل تنظيم المشروع:
تم تنظيم الملفات والمجلدات بدقة حسب الهيكل المطلوب في التكليف:
* `index.html`
* `README.md`
* `css/style.css`
* `js/main.js`
* `images/`
